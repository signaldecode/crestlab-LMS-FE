강의 플랫폼의 카테고리가 **주식 / 가상 자산 / 부동산** 3가지로 확정되었다.

**Why:** 기존 범용 강의 플랫폼 컨셉에서 재테크/투자 교육 플랫폼으로 컨셉이 전면 변경됨.

**How to apply:** 카테고리 네비게이션, 필터, data JSON, 홈 섹션 구성, SEO 메타 등 모든 곳에서 이 3개 카테고리를 기준으로 설계한다. 범용 교육 플랫폼 가정 대신 투자/재테크 도메인에 맞는 용어와 구조를 사용할 것.
